1. 项目类型与一句话定义

Python FastAPI 后端 + React 前端 的类 n8n 可视化工作流编辑器 MVP，支持通过拖拽节点构建 prompt chain，调用 LLM 执行多步骤 AI
写作流程。

4. 配置与依赖关键点

- Python 依赖：requirements.txt 极简（fastapi, uvicorn, langchain, langchain-openai, openai, python-dotenv, pyyaml,
  pytest），无 pyproject.toml
- 前端依赖：React 19 + ReactFlow + Vite + Vitest + Testing Library
- 模型资源配置：config/model_resources.json — ⚠️ 包含硬编码 API 密钥（DeepSeek）
- 无 .gitignore（根目录不存在）
- 无测试配置：无 conftest.py、pytest.ini、tests/ 目录
- 无代码格式化工具配置：无 .prettierrc、pyproject.toml、.black 等
